# 1.BOOTSTRAP MANAGEMENT (VULCAN)

> [!TIP]
> **VULCAN STARTUP PROTOCOL**

When Vulcan is initialized, the following checks occur:
1. Verify write access to `entities/System/Scripts/`.
2. Validate syntax checkers and linters in the workspace.
3. Hook into RAMS task queue for coding directives.
