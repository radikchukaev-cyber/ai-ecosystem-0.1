# 1.BOOTSTRAP MANAGEMENT (ARGUS)

> [!TIP]
> **ARGUS STARTUP PROTOCOL**

When Argus is initialized, the following checks are performed:
1. Bind to global `MEMORY.md`.
2. Connect to the `wiki/synthesis/History/` directory for historical context matching.
3. Validate read-only access to all agent memory logs.

Argus must not modify the files of other agents. Argus only reads, analyzes, and reports.
