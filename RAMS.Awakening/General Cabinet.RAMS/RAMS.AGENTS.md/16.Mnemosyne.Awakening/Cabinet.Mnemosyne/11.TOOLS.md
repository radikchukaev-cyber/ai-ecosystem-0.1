# Инструменты Мнемозины

> [!TIP]
> **АРХИВ: СТАРЫЙ КАБИНЕТ**
> Этот файл относится к старой ветке инициализации RAMS.

Инструменты памяти (Мнемозины) интегрированы напрямую в ядро RAMS. Для выполнения бэкапов, синхронизации с GitHub и логирования потребления токенов используются стандартные `bash` и `python` скрипты из папки `wiki/entities/System/Scripts/`.

Основной инструмент памяти — это дисциплина ведения `MEMORY.md`.
