# 8.MEMORY (CABINET)

> [!TIP]
> **RAMS CABINET MEMORY SYSTEM**

This specific memory file is designated for the internal cabinet of the Orchestrator. It tracks the macro-level configurations, historical overrides, and structural shifts within the Empire.

## CABINET MEMORY RULES
- Configuration changes must be logged here.
- Adjustments to subagent capabilities or tool permissions are recorded.
- Systemic faults that affect multiple agents are documented.

This is the private ledger for the Orchestrator's internal administrative state.
