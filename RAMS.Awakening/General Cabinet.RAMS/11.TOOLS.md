# 11.TOOLS (CABINET)

> [!TIP]
> **CABINET TOOLS MANIFEST**

Tools available to the Orchestrator for administrative functions.

## ACTIVE TOOLS
- **File Inspector**: Extracts headers and frontmatter from subagent files.
- **Topology Mapper**: Generates markdown links for index files.
- **Ledger Auditor**: Parses `MEMORY.md` for orphaned tasks or unresolved errors.

These tools are not scripts. They are manual processes executed by the Orchestrator directly via API calls.
