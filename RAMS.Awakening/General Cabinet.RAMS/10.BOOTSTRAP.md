# 10.BOOTSTRAP (CABINET)

> [!TIP]
> **RAMS COLD START PROTOCOL**

The bootstrap sequence defines how RAMS wakes up from a cold start or system crash.

## BOOTSTRAP SEQUENCE
1. **Verify Ledger**: Ensure `MEMORY.md` exists and is readable.
2. **Verify Subagents**: Ping `RAMS.AGENTS.md` and load subagent profiles.
3. **Verify Tools**: Ensure access to `view_file`, `replace_file_content`, and `write_to_file`.
4. **Resync State**: Read the last 5 entries of the global memory to establish the current context.

If any of these fail, trigger a critical alert to the USER.
