# 📁 Банк Шаблонов — wiki/templates/

Эта папка содержит **официальные шаблоны** экосистемы CARPATH.  
Все новые документы должны создаваться **только на основе шаблонов** из этой директории.

---

## Зачем это нужно

Шаблоны обеспечивают:
- **Единообразие** структуры документов по всей вики
- **Правильную привязку** к индексам через стандартные отметки
- **Скорость** — агент не придумывает структуру с нуля, а заполняет готовую форму
- **Читаемость** — любой агент или человек мгновенно ориентируется в незнакомом файле

## Инструкция по использованию

### Шаг 1 — Скопируй нужный шаблон

Найди подходящий шаблон в этой папке и скопируй его содержимое в новый файл в нужной директории.  
**Не редактируй сами шаблоны.** Они — мастер-копии.

### Шаг 2 — Переименуй файл

Дай файлу осмысленное имя по следующей конвенции:

```
НАЗВАНИЕ_СУЩНОСТИ_ТИП.md
```

Примеры:
- `CARPATH_COMPANY.md`
- `VULCAN_MEMORY_LOG.md`
- `LAUNCH_Q3_TIMELINE.md`
- `GENERATE_REPORT_SKILL.md`

### Шаг 3 — Заполни поля

Замени все плейсхолдеры в квадратных скобках `[...]` реальными данными.  
Удали инструктивные блоки `> Шаблон ... удали этот блок`.

### Шаг 4 — Привяжи к нужному индексу

Убедись, что строка `**Отметки:**` в конце файла ведёт на правильный индексный файл.  
Добавь ссылку на новый документ в соответствующий `*_INDEX.md`.

---

## Правила

- ❌ **Не создавай документы без шаблона** — это порождает хаос в структуре
- ❌ **Не изменяй файлы в этой папке** без согласования с VULCAN
- ✅ **Если шаблона не хватает** — запроси создание нового у VULCAN через Оркестратора
- ✅ **Отметки в конце файла — обязательны** и должны вести на релевантный индекс
