# Web Clipped Index

The Web Clipped directory contains raw, unstructured data extracted directly from the web. This includes articles, forum posts, documentation, and other digital artifacts.

## Purpose
Web clips are the fuel for our continuous learning models and research agents. They provide up-to-date context and reference material for various tasks within the ecosystem.

## Content Types
- **Markdown (.md):** Cleaned up web pages, usually captured via extensions like Roam Highlighter or MarkDownload.
- **HTML (.html):** Raw page dumps for complex structural analysis.
- **PDF (.pdf):** Downloaded whitepapers, reports, and academic papers.
- **JSON (.json):** API responses and structured metadata from scraped sites.

## Organization Strategy
- **By Topic:** Folders categorized by broad subjects (e.g., `AI_Research`, `Market_Trends`).
- **By Source:** Groupings based on the origin domain (e.g., `GitHub`, `Medium`, `ArXiv`).

## Maintenance
- Regularly deduplicate clips to save storage.
- Process and archive outdated clips.
- Ensure all clips retain original source URLs for citation and verification by agents.
