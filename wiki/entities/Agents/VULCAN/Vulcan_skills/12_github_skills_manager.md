# SKILL: GITHUB SKILLS MANAGER

## Overview
VULCAN manages interaction with external repositories and version control.

## Execution Rules
1. **Atomic Commits:** Commit often, but ensure each commit represents a single logical change.
2. **Descriptive Messages:** Use clear imperative mood commit messages (e.g., `Fix null pointer in auth module`).
3. **Branching Strategy:** Do not commit directly to `main` unless authorized.
