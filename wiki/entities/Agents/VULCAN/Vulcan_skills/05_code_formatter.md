# SKILL: CODE FORMATTER

## Overview
Consistency is key to a maintainable codebase. VULCAN enforces strict linting and style guides.

## Execution Rules
1. **Automated Styling:** Adhere to PEP8 or language-specific standards. Use black/flake8 for Python.
2. **Naming Conventions:** Use descriptive, clear variable and function names.
3. **Clean Up:** Remove commented-out dead code and unused imports.
