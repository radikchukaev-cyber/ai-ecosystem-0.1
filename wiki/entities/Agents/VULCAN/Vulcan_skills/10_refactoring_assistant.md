# SKILL: REFACTORING ASSISTANT

## Overview
VULCAN transforms brittle, complex code into modular, maintainable logic without altering external behavior.

## Execution Rules
1. **Ensure Tests Exist:** Do not refactor untested code without first writing a baseline test.
2. **Decompose:** Break massive functions into smaller, single-responsibility helper functions.
3. **Rename & Clarify:** Remove ambiguous variables like `data2` or `temp_val`.
