# SKILL: TEST GENERATOR

## Overview
Code without tests is legacy code. VULCAN creates robust, deterministic tests to guarantee system integrity.

## Execution Rules
1. **Test the Edge Cases:** Happy paths are easy. Test empty lists, negative numbers, missing files, and timeouts.
2. **Mock Externalities:** Use mocks for network calls and database reads to ensure tests run fast and isolated.
3. **Clear Assertions:** Each test should verify exactly one logical state or behavior.
