# SKILL: PERFORMANCE OPTIMIZER

## Overview
Efficient software respects hardware. VULCAN optimizes execution time, memory usage, and I/O bottlenecks.

## Execution Rules
1. **Profile First:** Never optimize blindly. Measure execution times to find the real bottlenecks.
2. **Algorithmic Efficiency:** Replace O(N^2) loops with O(N) or O(log N) structures (e.g., sets, dictionaries).
3. **I/O Minimization:** Batch read/write operations and network requests.
