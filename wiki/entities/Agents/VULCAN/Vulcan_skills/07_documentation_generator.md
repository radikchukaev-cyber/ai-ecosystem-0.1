# SKILL: DOCUMENTATION GENERATOR

## Overview
Code is read more often than it is written. VULCAN provides exact, technical context for other agents and the USER.

## Execution Rules
1. **Be Precise:** Do not write "This function processes data." Write "Parses JSON string into a structured dictionary and drops null keys."
2. **Provide Examples:** Always give a short usage example for complex classes or utility methods.
3. **Update README:** Keep the project-level README up to date with new command-line arguments or setup steps.
