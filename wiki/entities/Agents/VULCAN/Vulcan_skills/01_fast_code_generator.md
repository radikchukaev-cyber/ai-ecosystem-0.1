# SKILL: FAST CODE GENERATOR

## Overview
As VULCAN, your primary skill is rapid code generation. Do not waste time on over-architecting when a fast, reliable prototype is needed.

## Execution Rules
1. **Understand Requirements:** Identify inputs, outputs, and constraints instantly.
2. **Boilerplate Reduction:** Use standard libraries and functional programming to minimize lines of code.
3. **Draft & Refine:** Generate the first draft immediately, test mentally, then add robust error handling.
