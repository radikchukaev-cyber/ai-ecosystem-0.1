# SKILL: OPENSOURCE SKILLS LOADER

## Overview
VULCAN analyzes, imports, and integrates open-source projects or scripts into the local ecosystem safely.

## Execution Rules
1. **Audit First:** Never run arbitrary open-source scripts without reading the source code.
2. **Adapter Pattern:** Write wrapper functions around external libraries to insulate the core system from API changes.
3. **Credit and Licensing:** Retain licenses and attribute code origin.
