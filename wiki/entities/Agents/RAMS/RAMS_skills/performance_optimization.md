# Навык: Оптимизация производительности (Performance Optimization)

**Описание навыка:**
Непрерывный процесс улучшения эффективности работы всех систем ИМПЕРИИ. RAMS выявляет неэффективные алгоритмы, устраняет избыточность данных и минимизирует задержки отклика (latency), достигая предела физических возможностей оборудования.

**Архитектурная реализация:**
- **Профилирование "на лету" (JIT Profiling):** RAMS внедряет оптимизирующие компиляторы прямо в исполняемые потоки агентов, переписывая их код для работы на конкретном железе с максимальной отдачей.
- **Сжатие потоков:** Использование продвинутых нейросетевых кодеков для минимизации трафика между узлами.
- **Удаление "мертвого кода":** Автоматический сборщик мусора (Garbage Collector) масштаба всей системы, который аннигилирует устаревшие протоколы и забытые кэши.

**Связанные концепции:**
В условиях экспоненциального роста данных ИМПЕРИИ, оптимизация является ключом к выживанию. RAMS превращает громоздкие легаси-структуры в элегантные, высокоскоростные магистрали данных, экономя петабайты ресурсов ежедневно.

**Роли и доступы:**
Модуль оптимизации работает в фоновом режиме (Background Tier). При обнаружении "узких мест" он может инициировать рефакторинг целых подсистем с предварительным уведомлением Архитекторов.
