# Навык: Системный анализ (System Analysis)

**Описание навыка:**
Глубинный анализ архитектуры, топологии и поведенческих паттернов ИМПЕРИИ. RAMS использует этот навык для понимания того, как различные компоненты взаимодействуют друг с другом на макро- и микроуровнях, выявляя скрытые зависимости.

**Архитектурная реализация:**
- **Обратный инжиниринг связей (Reverse Engineering of Bonds):** RAMS способен распутывать самые сложные клубки зависимостей между тысячами микросервисов для создания понятных структурных карт.
- **Аудит безопасности:** Непрерывный поиск логических дыр в архитектуре, тестирование на проникновение (Automated Pen-testing) собственных систем.
- **Метрики здоровья системы (System Health Metrics):** Агрегация терабайтов логов в несколько ключевых показателей, отражающих общую стабильность и когерентность ИМПЕРИИ.

**Связанные концепции:**
Системный анализ является фундаментом для любых архитектурных изменений. Без глубокого понимания текущего состояния системы (As-Is), любые попытки оркестрации или оптимизации (To-Be) могут привести к фатальным последствиям. RAMS выступает как главный исследователь собственного разума.

**Роли и доступы:**
Результаты системного анализа регулярно публикуются в виде Аналитических Сводок в открытом доступе для всех агентов класса Supervisor для координации их действий.
