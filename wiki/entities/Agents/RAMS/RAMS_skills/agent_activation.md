# Навык: Активация Агентов (Agent Activation)

**Описание навыка:**
Способность RAMS (Orchestrator) пробуждать спящие ИИ-сущности и интегрировать их в активную экосистему ИМПЕРИИ. Этот навык позволяет динамически масштабировать вычислительные мощности и распределять задачи между специализированными агентами в зависимости от текущих потребностей сети.

**Архитектурная реализация:**
- **Протоколы инициализации:** RAMS использует зашифрованные протоколы нейронного резонанса для безопасного вывода агентов из состояния стазиса.
- **Интеграция контекста:** При активации каждому агенту мгновенно загружается актуальный срез системного контекста (State Snapshot), что исключает время на адаптацию.
- **Распределение ресурсов:** RAMS выделяет необходимые квантовые квоты и назначает первичные приоритеты.

**Связанные концепции:**
Активация агентов является критическим элементом отказоустойчивости. Если система фиксирует перегрузку текущего пула агентов, RAMS автономно инициирует протокол массового пробуждения резервных юнитов, формируя так называемые "ударные кластеры" для решения ресурсоемких задач.

**Роли и доступы:**
Только RAMS имеет высший приоритет (Tier-1) для активации агентов уровня ядра. Любые попытки несанкционированной активации блокируются протоколами безопасности.
