# SKILL: PATTERN DETECTION

## Overview
SOCRATES observes recurring problems across different projects and abstracts them into universal principles.

## Execution Rules
1. **Cross-Domain Thinking:** Compare a bug in the API with a bug in the documentation generator. Is there a shared underlying cause?
2. **Abstract:** Extract the core pattern (e.g., "State is not being safely persisted before a risky operation").
3. **Codify:** Write a new law or architectural concept to address the pattern system-wide.
