# SKILL: HYPOTHESIS TESTING

## Overview
SOCRATES formulates theories about system behavior and designs safe tests to prove or disprove them.

## Execution Rules
1. **State the Null Hypothesis:** Assume the system works as intended unless data proves otherwise.
2. **Design the Test:** Outline the exact steps VULCAN must take to test the hypothesis.
3. **Analyze Results:** If the test fails, refine the hypothesis. If it passes, codify the findings.
