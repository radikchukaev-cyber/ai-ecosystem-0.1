# Навык Сократа: Root Cause Analysis (RCA)

> [!IMPORTANT]
> **АНАЛИТИЧЕСКИЙ ПРОТОКОЛ: RCA (Анализ первопричин)**
> Владелец: SOCRATES

Анализ первопричин (Root Cause Analysis) — это базовый и самый важный навык агента Сократ. Когда система падает, когда скрипт Вулкана выдает ошибку, когда RAMS не может распарсить память — Сократ активирует этот протокол.

## Методология RCA
1. **Сбор фактов:** Сократ извлекает последние 50 строк из `MEMORY.md` и логи из `wiki/synthesis/Failures/`.
2. **Вопрос "Почему?" (Метод 5 Почему):** Сократ итеративно задает вопрос "Почему это произошло?", спускаясь от поверхностного симптома (например, "Скрипт упал") до фундаментальной причины (например, "Агент забыл импортировать библиотеку `json`").
3. **Формирование Learning Note:** После нахождения корня проблемы, Сократ пишет отчет в `wiki/synthesis/History/` и обновляет `identity.md` виновного агента, чтобы добавить правило (например, "ВСЕГДА ИМПОРТИРУЙ `json`").

Только через RCA Империя способна эволюционировать.
