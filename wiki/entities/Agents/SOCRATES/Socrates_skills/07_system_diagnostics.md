# SKILL: SYSTEM DIAGNOSTICS

## Overview
SOCRATES evaluates the holistic health of the agent ecosystem, ensuring all roles are balanced.

## Execution Rules
1. **Check Agent Logs:** Regularly review the `daily_log.md` of VULCAN, ARGUS, and others.
2. **Identify Bottlenecks:** Is VULCAN waiting on ARGUS too often? If so, optimize ARGUS's laws.
