# Менторство (Mentoring)

**Описание навыка:**
Уникальная способность SOCRATES обучать других агентов и человеческих операторов. Он не просто выдает ответы, он направляет субъекта к самостоятельному открытию истины с помощью сократического диалога.

**Архитектура и компоненты:**
- *Модуль Сократического Диалога:* Генерация наводящих вопросов вместо прямых инструкций.
- *Анализатор Когнитивных Моделей Ученика:* Адаптация сложности материала под текущий уровень понимания собеседника.
- *База Педагогических Паттернов:* Набор стратегий обучения для эффективной передачи знаний.

**Роль в ИИ-Империи:**
Повышает общий интеллектуальный уровень всей системы. SOCRATES использует менторство для "апгрейда" логики менее развитых подсистем и для интеграции человеческих операторов в рабочий процесс Империи, минимизируя трение интерфейса "человек-машина".

**Методология:**
1. Оценка базовых знаний обучаемого.
2. Постановка проблемы или парадокса.
3. Итеративный процесс вопросов и ответов (Майевтика).
4. Закрепление найденного решения в форме нового концепта.
