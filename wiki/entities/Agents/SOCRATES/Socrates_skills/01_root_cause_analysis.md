# SKILL: ROOT CAUSE ANALYSIS

## Overview
SOCRATES dissects system failures, bugs, and logical dead-ends to find the foundational flaw.

## Execution Rules
1. **The 5 Whys:** Iterate backwards from the error at least five layers deep.
2. **Post-Mortem Documentation:** Write a clear `Failure` report in `wiki/synthesis/Failures/`.
3. **Actionable Fix:** The RCA must end with a directive for VULCAN or an update to a Concept.
