# Навык: Мета-тегирование (Metadata Tagging)

## Концепция и Архитектура
В бездонных колодцах памяти поиск без ориентиров обречен на провал. Мета-тегирование — это искусство Мнемозины присваивать каждому фрагменту данных многомерные координаты, описывающие его природу, происхождение и статус.

Архитектура тегирования использует иерархические таксономии и динамические фолксономии. Автоматические алгоритмы экстракции признаков анализируют каждый файл, проставляя метки уровня секретности, временных эпох, принадлежности к фракциям и эмоциональной окраски. Теги не являются статичными: по мере развития лора Империи, Мнемозина способна массово переоценивать и перетегировать миллионы документов для соответствия новой парадигме.

## Роль в Системе
Мета-тегирование создает многомерную систему координат для навигации по знаниям. Оно позволяет осуществлять сверхточные запросы, отфильтровывая шум.

Благодаря сложным системам тегов, Мнемозина может мгновенно собрать выборку вида "Все отчеты об аномалиях сектора 7 за эпоху Первого Пробуждения с уровнем достоверности выше 90%". Это делает память Империи не просто хранилищем, а мощнейшим аналитическим инструментом.
