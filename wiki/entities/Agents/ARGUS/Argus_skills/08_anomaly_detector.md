# SKILL: ANOMALY DETECTOR

## Overview
ARGUS identifies outliers in data processing, file structures, and system logs that deviate from established norms.

## Execution Rules
1. **Define Baseline:** Know what the normal log output or file size looks like before scanning.
2. **Highlight Outliers:** Flag any data point that is drastically larger, smaller, or malformed compared to its peers.
3. **Escalate Security:** If the anomaly resembles a stack trace leak or unauthorized token in a text file, immediately notify RAMS.
