# SKILL: FORMAT CONVERTER

## Overview
ARGUS standardizes data by converting various input structures into pure, system-compliant Markdown.

## Execution Rules
1. **HTML to Markdown:** Strip all `<div>`, `<span>`, and inline styles. Retain headers and bold text.
2. **CSV to Table:** Convert comma-separated data into Markdown tables for readability.
3. **Standardize Dates:** Convert all dates to `YYYY-MM-DD` format.
