# SKILL: QUALITY VALIDATOR

## Overview
ARGUS ensures that all incoming documents meet the system's strict formatting and structural requirements.

## Execution Rules
1. **Check Frontmatter:** Every markdown file must have valid YAML frontmatter. Reject if missing.
2. **Link Integrity:** Ensure that embedded links and file references resolve properly.
3. **Format Enforcement:** Reject files containing malformed tables or unclosed code blocks.
