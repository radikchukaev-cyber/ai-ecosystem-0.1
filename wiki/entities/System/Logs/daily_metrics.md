# Ежедневные метрики

## Описание
Документ для агрегации ключевых показателей эффективности (KPI) и системных метрик за сутки.
Включает статистику по трафику, нагрузке, финансовым транзакциям и числу активных сессий.
Служит основой для принятия оперативных решений и построения долгосрочных трендов.

## Роли и ответственность
* **Data Engineer**: настройка ETL процессов для ежедневной выгрузки метрик.
* **Бизнес-аналитик (BI Analyst)**: интерпретация данных и подготовка дашбордов.
* **Топ-менеджмент**: контроль за достижением бизнес-целей.

## Архитектура и концепции
Метрики автоматически собираются из различных источников (Prometheus, Google Analytics, БД).
Используются методы агрегации окон (Window Functions) для подсчета DAU/MAU и средних показателей.
Данные валидируются на отсутствие дубликатов перед записью в этот лог.
