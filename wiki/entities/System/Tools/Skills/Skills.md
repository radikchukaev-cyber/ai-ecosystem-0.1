# 🎯 TOOLS SKILLS

Эта директория представляет собой динамическую библиотеку расширений, подключаемых модулей (MCP - Model Context Protocol) и специализированных скиллов, которые радикально расширяют базовые возможности наших LLM-агентов. Если агенты — это "мозг", то скиллы — это их "руки", позволяющие взаимодействовать с внешним миром.

### Архитектура Скиллов

Архитектура системы построена по модульному принципу (Plug-and-Play). Каждый скилл здесь изолирован и выполняет единственную, хорошо определенную функцию (Unix-way).
- **Парсеры и Конвертеры:** Инструменты для безопасного извлечения данных (например, `json-parser-safe`), преобразования форматов, парсинга веб-страниц и обработки медиа.
- **Интеграторы API (API Wrappers):** Скрипты, инкапсулирующие логику общения с внешними сервисами (YouTube, GitHub, Telegram), управляющие rate-лимитами и аутентификацией.
- **Внутренние утилиты (System Utils):** Модули для работы с локальной файловой системой, семантического векторного поиска (`vector-search`) и взаимодействия с Obsidian CLI.

### Принципы Использования и Интеграции

1. **Регистрация:** Любой новый скилл должен быть в обязательном порядке задокументирован и внесен в глобальный реестр `TOOLS_INDEX.md`, чтобы агенты-оркестраторы могли "узнать" о его существовании и добавить в свой контекст.
2. **Инкапсуляция и Безопасность:** Скилл не должен содержать хардкодных секретов (они подтягиваются из `Root_Configs`). Все входящие параметры должны строго валидироваться (Type Checking).
3. **Обработка Ошибок (Graceful Degradation):** Каждый инструмент должен возвращать понятные сообщения об ошибках, чтобы агент мог самостоятельно проанализировать причину сбоя и исправить свой запрос.
4. **Тестирование:** Перед развертыванием в продакшен, скилл тестируется на изолированных данных. Агент VULCAN часто использует эти модули как кирпичики для построения более сложных автоматизированных цепочек.

Грамотное управление скиллами позволяет бесконечно масштабировать возможности "Карпат", не усложняя при этом базовую логику самих агентов.
