# ▶️ YOUTUBE API WRAPPER

Скилл `youtube-api-wrapper` представляет собой специализированный адаптер для защищенного и эффективного взаимодействия системы "Карпаты" с экосистемой YouTube (YouTube Data API v3 и неофициальными библиотеками для извлечения субтитров). Это "глаза и уши" системы в мире видеоконтента.

### Архитектура Интеграции

Инструмент изолирует сложную логику работы с сервисами Google от бизнес-логики агентов:
- **Квотирование (Rate Limiting & Quotas):** API Google имеет строгие лимиты. Обертка отслеживает количество потраченных токенов в сутки, кеширует результаты (чтобы не запрашивать одно и то же видео дважды) и ставит запросы в очередь.
- **Извлечение Транскриптов:** Основная ценность — умение "вытаскивать" субтитры (включая автогенерируемые) из видео, обходя региональные блокировки или капчи (используя локальные скрипты или прокси).
- **Парсинг Метаданных:** Извлечение названия, описания, тегов, комментариев и таймкодов, которые затем аккуратно конвертируются в Markdown-заметку.

### Использование и Применение

1. **Анализ Контента:** Агент `YouTube_Researcher` использует этот скилл для "просмотра" видео. Получив ссылку, он выкачивает транскрипт, скармливает его LLM и формирует краткую выжимку (Summary), выделяя ключевые инсайты.
2. **Генерация Конспектов:** Автоматическое превращение полезных лекций, подкастов или туториалов в структурированные статьи в базе знаний Шефа, дополненные ссылками на конкретные таймкоды.
3. **Мониторинг:** Возможность по расписанию проверять определенные каналы или поисковые запросы, автоматически загружая новые видео по заданным тематикам.
4. **Аудио-резервирование:** В связке с другими инструментами (например, yt-dlp), скилл может инициировать скачивание аудиодорожки для локального распознавания через Whisper, если стандартные субтитры отсутствуют.

Эта обертка экономит десятки часов времени, позволяя потреблять видеоконтент в текстовом виде со скоростью чтения.
