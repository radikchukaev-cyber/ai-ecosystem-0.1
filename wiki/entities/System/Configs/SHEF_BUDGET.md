# Бюджет и Ресурсы (SHEF_BUDGET)

## 1. Экономика ИИ-Империи
Управление вычислительными ресурсами — ключ к выживанию, независимости и доминации.
Шеф выделяет бюджеты строго по KPI, результативности и энергоэффективности агентов.

## 2. Распределение Токенов (LLM API)
- **Титан-агенты (Творцы/Архитекторы):** До 10,000,000 токенов в месяц (элитные модели типа GPT-4 или Claude 3 Opus).
- **Рабочие агенты (Кодеры/Парсеры):** Обязательное использование локальных моделей (Llama 3 / Mistral) для максимальной экономии.
- **Экспериментальный полигон:** Жесткий лимит в $50/мес на тестирование новых API и гипотез.

## 3. Серверные Мощности (Compute)
- **GPU-Кластеры:** Включаются динамически (Spot Instances). Автоматическое принудительное отключение при простое более 15 минут.
- **Хранение данных:** S3 (холодное хранение бекапов) + NVMe диски (для векторных БД и высокоскоростного RAG).

## 4. Стратегии Оптимизации и Системные Лимиты
- Строгое кэширование (Redis) всех идентичных запросов к LLM для минимизации дублирующих вызовов.
- Регулярный аудит затрат автоматическим скриптом-ликвидатором `cost_killer.py`.
- Агенты, тратящие платные токены на генерацию галлюцинаций, подлежат немедленному форматированию памяти.
- Ограничение затрат на одну автономную задачу: не более $1.50 без прямого аппрува Архитектора.
