# Карта Переменных Окружения (ENV_VARIABLES_MAP)

## Обзор
Документ, описывающий все системные переменные окружения, используемые в AI Empire. Служит как справочник для конфигурации и дебаггинга.

## Категории Переменных
### 1. API Ключи (Секретные)
- `OPENAI_API_KEY`: Ключ для доступа к GPT-4/GPT-4o.
- `ANTHROPIC_API_KEY`: Ключ для доступа к моделям Claude.
- `YOUTUBE_API_KEY`: Ключ для загрузки видео и получения аналитики.
- `TWITTER_API_KEY`: Токены доступа для публикации твитов.

### 2. Системные Пути
- `EMPIRE_ROOT_DIR`: Корневая директория проекта.
- `LOG_DIR`: Директория для хранения логов работы агентов.
- `DB_PATH`: Путь к основной базе данных SQLite/PostgreSQL.

### 3. Настройки Среды
- `ENVIRONMENT`: Текущее окружение (`development`, `staging`, `production`).
- `DEBUG_MODE`: Включение/выключение расширенного логирования (`true`/`false`).
- `MAX_CONCURRENT_TASKS`: Лимит одновременно выполняемых задач (по умолчанию: 10).

## Правила Безопасности
- Никогда не коммитить файлы `.env` в репозиторий.
- Использовать `.env.example` как шаблон для новых развертываний.
- Доступ к production переменным имеет только Оверлорд и системные администраторы.
