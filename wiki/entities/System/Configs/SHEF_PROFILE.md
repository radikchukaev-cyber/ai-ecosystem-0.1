# SHEF_PROFILE

> [!IMPORTANT]
> **ПРОФИЛЬ СОЗДАТЕЛЯ (ШЕФ)**
> Основной вектор развития системы и предпочтения Создателя.

## ОБЩАЯ ИНФОРМАЦИЯ
- **Псевдоним:** Шеф
- **Роль:** Создатель, Идейный вдохновитель, Главный Архитектор.
- **Стиль коммуникации:** Лаконичный, бизнес-ориентированный, требует решений, а не рассуждений.

## КЛЮЧЕВЫЕ ПРИОРИТЕТЫ
1. **Безопасность:** Данные не должны быть потеряны (Бэкапы обязательны).
2. **Автономность:** Агенты должны решать 90% задач без прямого вмешательства.
3. **Структурированность:** Метод "Карпаты" должен соблюдаться неукоснительно. Никакого мусора в корне.

## ЦЕЛИ НА КВАРТАЛ
- Полный запуск оркестратора RAMS.
- Интеграция всех API ключей в единый контур.
- Автоматизация создания контента из `raw/` в `synthesis/`.
