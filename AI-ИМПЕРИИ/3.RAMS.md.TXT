## ПРОБЛЕМА
Скрипт упал при парсинге API-ответа

## ЛОГИРОВАНИЕ
JSONDecodeError: Expecting value: line 1 column 1
File: wiki/entities/System/Scripts/parse_api.py, line 45
Response: "<!DOCTYPE html>..." (HTML ошибка 500)

## ROOT CAUSE ANALYSIS
API вернула HTML (ошибка 500) вместо JSON.
Код не проверил status_code перед парсингом.

## КАК ИСПРАВИЛИ
```python
if response.status_code != 200:
    logging.error(f"API ошибка: {response.status_code}")
    return None

try:
    data = json.loads(response.text)
except json.JSONDecodeError as e:
    logging.error(f"JSON ошибка: {e}")
    raise
```
