# 📚 RAW SOURCES

## 📌 Назначение (Purpose)
Директория `sources` выступает в качестве резервного хранилища оригиналов и сырых внешних источников. Здесь складируются объемные референсные материалы, документация, академические статьи и мануалы, которые лежат в основе базы знаний.

## 🔄 Поток данных (Data Flow)
1. **Импорт:** Файлы импортируются в исходном виде до какой-либо обработки агентом ARGUS или другими аналитиками.
2. **Индексация:** Текстовые источники индексируются для семантического поиска (RAG), чтобы агенты могли ссылаться на них при генерации ответов.
3. **Выборка:** Отдельные параграфы или концепции могут быть скопированы и переработаны в атомарные заметки (Synthesis), при этом всегда оставляя ссылку на первоисточник из этой папки.

## ⚠️ Правила работы (Rules)
- **Целостность:** Файлы служат "единым источником истины" (Source of Truth) для внешней информации. Никакой редакторской правки внутри этих файлов не допускается.
- **Тяжелые файлы:** Здесь могут находиться крупные текстовые документы (документации на тысячи строк), поэтому их не следует грузить в контекст ИИ целиком без предварительного поиска (grep/RAG).
- **Резервирование:** Папка служит резервным хранилищем оригиналов на случай, если процесс дистилляции информации пройдет с ошибками и потребуется повторный парсинг.

## 💡 Примеры (Examples)
- Полная документация API в Markdown (`API_Reference_v2.md`).
- Книги и мануалы, конвертированные в текст (`DeepLearningBook_ch1.md`).
- Сохраненные длинные лонгриды и статьи (`medium_architecture_article.md`).
