Здесь Вулкан фиксирует текущие спринты и бэклог задач по написанию скриптов. Каждое утро (после команды Шефа или RAMS) Вулкан проверяет этот файл.

## Текущий Бэклог
1. Поддержание скриптов валидации структуры хранилища.
2. Создание новых инструментов для парсинга по запросу АРГУСА.
3. Оптимизация старых Python скриптов (рефакторинг) по замечаниям СОКРАТА.

*Все завершенные задачи переносятся в `wiki/synthesis/History/`.*
